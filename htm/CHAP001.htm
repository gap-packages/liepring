1 Preamble

Abstract: This package gives access to the database of Lie p-rings of order at most p7 as determined by Mike Newman, Eamonn O'Brien and Michael Vaughan-Lee, see NOV04 and OVL05, and it provides some functionality to work with these Lie p-rings.

Acknowledgements: The Lazard correspondence induces a one-to-one correspondence between the Lie p-rings of order pn and class less than p and the p-groups of order pn and class less than p. This package provides a function to evaluate this correspondence; this function has been implemented and given to us by Willem de Graaf.
